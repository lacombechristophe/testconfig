## Règle de travail recommandée

1. Les fichiers Excel Diskoov définissent le **prix commercial prospect** : matrice dimensionnelle, remise, options, pose et transport lorsque la formule est complète.
2. Le catalogue fournisseur AquaMaster reste la référence pour les **limites techniques**, les références produits, les dimensions maximales et les dépendances d'options.
3. Les bases d'achat, marges et prix fournisseur du catalogue AquaMaster restent des données de **rentabilité interne**, non visibles côté prospect.
4. Si une donnée commerciale est absente ou contradictoire dans Excel, le configurateur affiche **« sur devis »** au lieu d'inventer ou d'extrapoler un prix.
5. Les prix automatiques restent présentés comme des **estimations TTC 2026**, soumises à validation technique, tant que Xavier n'a pas qualifié une règle comme contractuelle.
6. Le devis concurrent Pisceen/DERCYA reste exclu des prix et des règles Diskoov.
